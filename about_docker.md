# **3. 도커 명령어**   
## **도커 이미지 가져오기**   
```
 docker pull [이미지 이름]:[태그]
 ```
## **컨테이너 생성**   
```
 docker create [옵션] [이미지 이름]:[태그]
 ```
 ## **컨테이너 실행**   
```
docker start centos:7
 ```
 ## **컨테이너 목록**   
 ```
 docker ps
 ```
 ## **컨테이너 삭제**   
 ```
 docker rm [컨테이너 이름]
 ```
